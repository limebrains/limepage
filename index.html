PyDrill - Python Driver for Apache Drill

  • About
  • Download
  • Links

About

PyDrill is tool which enables communication with Apache Drill using Python.

It supports:

  • Python 2/3 compatibility,
  • Support for all rest API calls inluding profiles/options/metrics docs with full list.
  • Mapping Results to internal python types,
  • Compatibility with Pandas data frame,
  • Drill Authentication using PAM,

Install

You install PyDrill using following methods:

  • pip install pydrill
  • pip install git+git://github.com/PythonicNinja/pydrill.git
  • soon we will support more method!

Pet - Project Environment Tool

  • About
  • Download
  • Links

About

Pet is command line tool which helps you manage projects.

It supports:

  • Changing projects with autocompletion
  • Prepare shell on entering/exiting project
  • Running specific tasks within project
  • and much more!

Download

You can download pet using following methods:

  • bash -c "$(curl -fsSL https://raw.githubusercontent.com/limebrains/pet/master/install.bash)"
  • pip install https://github.com/limebrains/pet/archive/master.zip
  • git clone git@github.com:limebrains/pet.git && pip install -e pet
  • soon we will support more method!
